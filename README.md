1. 계산기 개요
이 프로그램은 교원(공무원)의 명예퇴직수당을 관련 법령의 일반 산식에 따라 **"예상"**해 보는 도구입니다. 2025년 교원 봉급표를 기준으로 하며, 실제 수령액과는 차이가 있을 수 있습니다.

2. 사용 방법 (단계별)
월봉급액 입력 (2가지 방법)

① 호봉 선택: [월봉급액 선택] 드롭다운 메뉴에서 본인의 호봉(50호봉부터 역순 정렬)을 선택합니다. 2025년 기준 본봉이 아래 입력란에 자동으로 채워집니다.

② 직접 입력: 또는, [직접 입력 가능] 란에 마우스로 클릭하여 예상 본봉을 숫자로만 입력할 수 있습니다. 입력 시 자동으로 쉼표(,)가 찍히고 하단에 한글 금액이 표시됩니다.

(참고: 호봉 선택 후 직접 입력을 수정하면, 드롭다운의 선택은 '--선택--'으로 초기화됩니다.)

정년 잔여월수 입력

명예퇴직을 신청하는 날(퇴직일)을 기준으로, 본인의 정년(만 62세)까지 남은 총 개월 수를 숫자로 입력합니다. (예: 1년이 남았다면 12 입력)

(참고: 명예퇴직은 법적으로 정년 잔여 기간이 1년(12개월) 이상 남은 경우에만 신청 가능합니다.)

'예상 수당 계산하기' 버튼 클릭

두 항목을 모두 입력한 뒤 버튼을 클릭합니다.

3. 결과 확인 (3단계)
버튼을 누르면 하단에 '예상 명예퇴직수당'과 함께 상세 계산 내역이 나타납니다.

[적용기준]

입력한 '정년 잔여월수'가 법령의 3가지 구간 (① 1~5년, ② 5~10년, ③ 10년 초과) 중 어디에 해당하는지 보여줍니다.

[산정산식]

[적용기준]에 따라 실제 계산에 사용된 3가지 공식 중 하나를 보여줍니다.

[계산 설명]

1. (A) 산정 기준액: 입력한 월봉급액에 65%를 곱한 값 (수당 계산의 기본금)입니다.

2. (B) 환산 월수: 입력한 '잔여월수'를 [산정산식]에 따라 변환한 '지급 배수'입니다.

3. ▶ 최종 수당: 위 1번과 2번을 곱한 최종 예상 수당(반올림)입니다.

4. [매우 중요] 반드시 읽어야 할 주의사항
이 계산기는 편의를 위해 제작된 단순 참고 자료이며, 아래와 같은 중요한 한계점을 가집니다.

🚨 2026년 퇴직자는 봉급표가 다릅니다.

이 계산기 속 봉급표는 2025년 기준입니다.

만약 2026년 2월 28일에 퇴직한다면, 명예퇴직수당은 2026년 1월 1일에 인상되는 "2026년 봉급표"를 기준으로 산정됩니다.

일반적으로 봉급이 인상되므로, 이 계산기 결과보다 실제 수령액은 더 높을 가능성이 큽니다.

⚠️ 단순 예상액이며, 법적 효력이 없습니다.

개인의 전체 근속연수, 징계 이력, 그 외 수당 내역 등은 전혀 반영되지 않았습니다.

💸 지급 상한액 및 예산 문제

법령에는 '지급 상한선' 규정이 있으며, 각 교육청의 예산 상황에 따라 실제 지급 여부나 대상자 선발이 달라질 수 있습니다.

✅ 최종 확인은 필수입니다.

가장 정확한 예상액은 **소속 학교 행정실(급여 담당자)**이나 **공무원연금공단(1588-4321)**을 통해 본인의 정보를 기준으로 상담받으셔야 합니다.
